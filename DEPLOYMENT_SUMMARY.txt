🎯 QUICK START LOCAL
================================================================================

1. Extrair pacote:
   unzip dashboard-rd-code-only.zip
   cd code-only-package

2. Adicionar dados (se code-only):
   cp ../historico_iqvia_202508.parquet .
   cp ../Preço.csv .

3. Instalar dependências:
   pip install -r requirements.txt

4. Rodar:
   streamlit run app.py

5. Abrir navegador:
   http://localhost:8501
